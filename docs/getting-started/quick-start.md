# Quick Start Guide

Get started with TrueCash in 5 minutes!

## First Launch

When you first launch TrueCash, you'll see the **Intro Screen** with a brief overview of the app's features.

1. **Swipe through** the intro slides or tap **SKIP** to go directly to the dashboard
2. **Grant notification permissions** (optional) to receive reminders for bills and budgets

## Adding Your First Transaction

### Income

1. Tap **"Add Income"** on the dashboard
2. Enter:
   - **Source**: e.g., "Salary"
   - **Amount**: e.g., 5000
   - **Date**: Select the date
3. Tap **Save**

### Expense

1. Tap **"Add Expense"** on the dashboard
2. Enter:
   - **Category**: e.g., "Groceries"
   - **Amount**: e.g., 150
   - **Date**: Select the date
   - **Note** (optional): e.g., "Weekly shopping"
   - **Tags** (optional): e.g., "food, essentials"
3. Tap **Save**

## Setting Up a Budget

1. Navigate to **Settings** â†’ **Budgets**
2. Tap **"Add Budget"**
3. Enter:
   - **Category**: e.g., "Dining Out"
   - **Monthly Limit**: e.g., 300
4. Tap **Save**

The dashboard will now show your budget progress!

## Creating a Saving Goal

1. Navigate to **Settings** â†’ **Saving Goals**
2. Tap **"Add Goal"**
3. Enter:
   - **Name**: e.g., "Emergency Fund"
   - **Target Amount**: e.g., 10000
   - **Current Amount**: e.g., 2000
4. Tap **Save**

Track your progress on the dashboard!

## Viewing Analytics

Tap the **Analysis** tab to see:

- **Spending Trends**: Monthly spending over time
- **Category Breakdown**: Where your money goes
- **Income vs Expenses**: Compare earnings and spending
- **Budget Performance**: How well you're sticking to budgets

## Customization

### Change Theme

1. Go to **Settings**
2. Tap **Theme**
3. Choose:
   - **System Default**
   - **Light Mode**
   - **Dark Mode**

### Change Currency

1. Go to **Settings**
2. Tap **Currency**
3. Select your preferred currency symbol

### Set Up PIN Lock

1. Go to **Settings**
2. Tap **Security**
3. Enable **PIN Lock**
4. Set your 4-digit PIN

## Backup & Restore

### Export Data

1. Go to **Settings** â†’ **Data Management**
2. Tap **Export Data**
3. Choose a location to save the JSON file

### Import Data

1. Go to **Settings** â†’ **Data Management**
2. Tap **Import Data**
3. Select your backup JSON file

!!! warning "Important"
    Importing data will **replace** all existing data. Make sure to export first!

## Tips for Success

1. **Track Daily**: Add expenses as they happen for accurate insights
2. **Review Weekly**: Check your analytics every week
3. **Set Realistic Budgets**: Start with achievable limits
4. **Use Tags**: Organize expenses with tags for better analysis
5. **Check AI Insights**: Review recommendations regularly
